## 🎯 Cara Pakai (Untuk Peserta Workshop)

1. **Download starter template** dari folder `starter/`
2. **Buka di Claude** (Claude.ai / Claude Code / Cursor)
3. **Copy-paste prompt** dari `prompts/workshop-prompts.md` satu per satu
4. **Ganti placeholder** dengan data diri kamu
5. **Deploy ke Netlify** — drag & drop folder, beres!
